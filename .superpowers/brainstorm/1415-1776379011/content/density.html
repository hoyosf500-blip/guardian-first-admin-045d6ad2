¿Qué densidad querés en las listas de pedidos?

Los 3 previews muestran la misma lista de pedidos pendientes con el estilo B (dark). La diferencia es cuánta info cabe en pantalla vs. cuánto espacio respira cada fila. Pensá en vos operando 8 horas seguidas.

A

Compacta

1 fila por pedido, datos esenciales en una línea.

~18 pedidos / pantalla · filas de 28px

#0001 Cliente A Producto demo $89.000 Bogotá · 2h Pend
#0002 Cliente B Producto demo $45.000 Medellín · 3h Pend
#0003 Cliente C Producto demo $120.000 Cali · 4h Pend
#0004 Cliente D Producto demo $68.000 Barranq. · 5h Pend
#0005 Cliente E Producto demo $34.500 Bucaram. · 6h Pend
#0006 Cliente F Producto demo $155.000 Pereira · 7h Pend

Para power users que escanean rápido. Tooltip al hover muestra detalle completo.
